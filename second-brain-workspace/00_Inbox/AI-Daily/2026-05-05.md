## 一、AI / 大模型 / 多模态 / Agent 前沿动态

### 1. OpenClaw — 个人 AI 助手框架
- **概述：** OpenClaw 是当下最火的个人 AI 助手框架，支持 Windows/macOS/Linux 全平台。近期社区生态爆发式增长，衍生出大量 Skills 和辅助工具（如 cc-switch、nanoclaw 等），被誉为"AI 龙虾"（logo 是龙虾 🦞）。截至今日已有超过 5400+ 社区 Skills。
- **来源：** https://github.com/openclaw/openclaw

### 2. NanoClaw — OpenClaw 的轻量容器化替代
- **概述：** 基于 Anthropic Agent SDK 构建的轻量级 AI 助手框架，运行在容器中保障安全。可以连接 WhatsApp、Telegram、Slack、Discord、Gmail 等消息应用，具备记忆和定时任务能力。适合需要安全隔离环境的用户。
- **来源：** https://github.com/qwibitai/nanoclaw

### 3. Paperclip — 零人工公司的开源编排框架
- **概述：** ★62k Stars 的开源项目实现"零人类公司"愿景——通过 AI Agent 编排实现完全自动化企业运营。代表了 AI Agent 从个人助理向企业级自动化演进的前沿方向。
- **来源：** https://github.com/paperclipai/paperclip

## 三、GitHub 新奇高 Star 项目

### 1. Caveman — 原始人语气编程 🪨
- **概述：** ★54k Stars！能把 Claude Code 的 token 用量砍掉 65% —— "why use many token when few token do trick" 💀 用原始人说话方式减少啰嗦输出，大幅节省推理成本。既好玩又实用。
- **来源：** https://github.com/JuliusBrussee/caveman

### 2. Awesome Design-MD — 品牌设计系统集合
- **概述：** ★71k Stars！收集了各大品牌设计系统的 DESIGN.md 文件。只要把任意一个 DESIGN.md 丢进项目，AI 就能生成对应的 UI。从 Apple、Google 到 Notion、Vercel 的风格任选，让 Vibe Coding 出来的界面不再丑。
- **来源：** https://github.com/VoltAgent/awesome-design-md

### 3. Graphify — 一键将代码/文档/图片转为知识图谱
- **概述：** ★43k Stars！Claude Code / Codex / Cursor 等编程助手的 Skill，能将任意文件夹（代码、SQL、脚本、文档、论文、图片、视频）转为可查询的知识图谱。App 代码 + 数据库 Schema + 基础设施全部在一个图谱里。
- **来源：** https://github.com/safishamsi/graphify

### 4. Nuwa-Skill — 蒸馏任何人的思维方式
- **概述：** ★17k Stars！中文项目。核心概念是"蒸馏任何人的思维方式"——心智模型、决策启发式、表达DNA。能从对话中提取一个人的思维特征并复现。创新项目。
- **来源：** https://github.com/alchaincyf/nuwa-skill
